1) Escribe un programa C con la estructura siguiente:

        int esPrimo(int n){

        }



        int main(){



        }

La función esPrimo debe devolver 1 si n es un numero primo, y 0, caso contrario. En la función main se debe 
pedir un número al usuario y escribir el número de primos indicados por la pantalla. Por ejemplo, si el usuario
 introduce el número 5, por la pantalla deberían salir los números 1,2,3,5,7 (el formato de salida no es importante).

2) Escribe un programa C que pida un número al usuario y devuelva el número más cercano que sea un cuadrado. 
Por ejemplo, si el usuario introduce 8, el programa debería escribir 9. Si el usuario introduce 18, el programa 
debería escribir 16. El programa debería funcionar para cualquier número que tenga entre 1 y 7 dígitos.

3) Escribe un programa C que pida al usuario un número positivo y calcule la suma de sus dígitos. Por ejemplo, 
si el usuario introduce 2784, el programa debería escribir en la pantalla 21. Si el usuario introduce 59, el 
usuario debería escribir 14. El programa debería funcionar para cualquier número que tenga entre 1 y 10 dígitos
