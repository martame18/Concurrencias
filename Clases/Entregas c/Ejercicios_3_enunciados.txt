Dada la definición de tipo

struct Persona{

   char nombre[30];

   int edad;

}



Implementa las funciones:

- void inicializa(struct Persona *p) que pide por el teclado un nombre y una edad y se la asigna a la variable struct Persona

- void escribe(struct Persona p) que escribe por la pantalla el nombre y la edad de la persona p

- void inicializaLista(struct Persona *lista, int m, int* n) que lee del teclado un número de personas y las almacena en el array lista. Se termina de leer cuando se introduce un valor especial que indique el final (por ejemplo, que la edad sea 0). El valor m es el tamaño del array lista y n es el número de personas que se han leído por teclado.

- void escribeLista(struct Persona *lista, int m) que escribe por la pantalla las m primeras componentes del array de personas lista.

- Utiliza las funciones anteriores en la función main para inicializar y escribir por la pantalla una lista de personas.
